Creating a chart

To create a chart, we need instantiate the Chart class. To do this, we need to pass in the 2d context of where we want draw the chart Here's an example.

<canvas id="myChart" width="400" height="400"></canvas>

//Get the context of the canvas element we want to select
var ctx = document.getElementById("myChart").getContext("2d");
var myNewChart = new Chart(ctx).PolaraArea(data);

We can also getthe contextof our canvas with jQuery. To do this we, this we need to get the DOM node out of the jQuery collection, and call the
getContext("2d") method of that.

// Get the context wiwh jQuery – using jQuerty's .get() method.
var ctx = $("#myChart") .get(0) .getContext("2d");
// This will get the first returned node in the jQuery collection
var myNewChart = new Chart(ctx);
